Alter

Layer.showAlert('有问题了啊'); //单引号内容可换，必填，窗口手动关闭。

操作成功提示框

Layer.showSucAlert(); //默认值格式代码，不填写内容时默认值为操作成功。窗口自动关闭

Layer.showSucAlert('小样的，终于成功了'); //单引号内容可换，可不填。窗口自动关闭

操作成功提示框+自定义提示内容+回调方法

Layer.showSucAlert('小样的，我终于成功了',showCall); //单引号内容可换，必填。窗口关闭后，有回调信息返回

function showCall(index)

{

layer.alert("我是回调奥"+index);

Layer.closeLayer(index);

}
操作失败提示框

Layer.showFailAlert(); //默认值格式代码，默认值为操作失败。窗口不自动关闭

Layer.showFailAlert('我失败了',0); //单引号内容可换，必填。窗口不自动关闭，参数为0，表示不自动关窗

Layer.showFailAlert('我失败了',1000); //单引号内容可换，必填。窗口不自动关闭，参数为1000(1秒)，表示1秒后自动关窗

OpenDialog【对话框、对话框+自定义宽、对话框+自定义宽高】

Layer.openDialog('我是Open窗口','#'); //单引号标题内容可换，必填，#号为链接地址。

注:此方法为top.layer.open弹窗，有放大缩小按钮

Layer.openDialog('我是Open窗口','#',30); //单引号标题内容可换，必填，#号为链接地址,30为宽，单位为%。

注:此方法为top.layer.open弹窗，有放大缩小按钮

Layer.openDialog('我是Open窗口','#',30,50); //单引号标题内容可换，必填，#号为链接地址,30为宽，50为高，单位为%。

注:此方法为top.layer.open弹窗，有放大缩小按钮

OpenDialog【对话框、对话框+自定义宽、对话框+自定义宽高】本系统弹窗使用的方法

Layer.openDialog1('我是Open窗口','#'); //单引号标题内容可换，必填，#号为链接地址。

注:此方法为layer.open弹窗，无放大缩小按钮

Layer.openDialog1('我是Open窗口','#',30); //单引号标题内容可换，必填，#号为链接地址,30为宽，单位为%。

注:此方法为layer.open弹窗，无放大缩小按钮

Layer.openDialog1('我是Open窗口','#',30,50); //单引号标题内容可换，必填，#号为链接地址,30为宽，50为高，单位为%。

注:此方法为layer.open弹窗，无放大缩小按钮

Message

Layer.showMsg('我只是简单的提示一下'); //简单的提示框，与官方的layer.msg操作一样。

Layer.showSucMsg('我是成功的提示框'); //与官方的操作一样，但本封装js自动增加了图标，直接添加显示内容即可，比官方简单一点。

Confirm【提示确认框】

Layer.showConfirm('你确定要这样操作吗',function(index){layer.alert('是的')});

提示内容及回调函数，与官方操作一样。

/** * 显示验证框 * @param {} content 提示内容 * @param {} yesFunction 确定以后的回调函数 function(index){layer.alert('是的')} */

showTips【提示框】

Layer.showTips();

未填写任何参数，默认提示内容为项目不详,弹窗指向ID为#parameter的输入框

Layer.showTips('请输入您的手机号码！');

显示已填写的内容请输入您的手机号码！,因未填写指定ID参数，弹窗指向ID为#parameter的输入框

Layer.showTips('请输入您的手机号码139XXXXXXXX！','#parameter1');

显示已填写的内容请输入您的手机号码139XXXXXXXX！,已填写指定ID参数，弹窗指向ID为#parameter1的输入框

load加载提示条

Layer.openDialogLoad(); //默认值格式代码，不填写内容时默认值为数据加载中，请稍候!。窗口自动关闭

Layer.openDialogLoad("数据疯狂加载中，你还是等等吧!"); //默认值格式代码，填写内容时显示为数据疯狂加载中，你还是等等吧!。窗口自动关闭
